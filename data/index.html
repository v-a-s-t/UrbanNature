Settings

With this page you can change your device's settings, such as which sensors are used, how often the device should post to the internet, and if it should use a WiFi connection.


WiFi

If you would like your device to use a WiFi connection, please select which WiFi it should connect to.


Adafruit IO

Provide your Adafruit IO username and key. These can be found in your Adafruit IO account.

Location

Optionally provide your device's latitude and longitude.


Sensors

Please select which sensors you'd like to record. You'll need to create and provide an Adafruit IO feed name for each sensor.
